# 🐧 Slime Siege

> **2D Endless Survival Game** developed with **Unity 6**.

## 📖 About

Slime Siege is a 2D survival game where the player controls a penguin warrior fighting endless waves of slimes. Every wave becomes more difficult, with boss enemies appearing at regular intervals.

The objective is simple:

- Survive as long as possible.
- Defeat enemies to earn score.
- Beat your highest score.

---

## 🎮 Features

- Endless wave system
- Boss waves
- Player health system
- Enemy health bars
- Score & Best Score saving
- Pause menu
- Story screen
- Options menu
- Controls menu
- Credits menu
- Sound effects
- Background music
- macOS & Windows builds

---
